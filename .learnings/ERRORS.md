# Errors

开发过程中遇到的错误和解决方案。

## 2026-03-16

### 错误 1: PowerShell 命令分隔符

**问题**: 使用 `&&` 连接命令时报错
```
标记"&&"不是此版本中的有效语句分隔符
```

**解决方案**: PowerShell 使用 `;` 而不是 `&&`
```powershell
cd path; npm command
```

### 错误 2: npm create 交互式确认

**问题**: npm create vitepress 需要交互式确认，进程被杀死

**解决方案**: 使用 PTY 模式并发送按键确认
```
发送：y + Enter
```

### 错误 3: 项目结构不完整

**问题**: 首次创建的项目结构不正确

**解决方案**: 删除后重新创建，手动完善目录结构

## 2026-03-16 - 首页表格样式混乱 (ERR-20260316-001)

**优先级**: high  
**来源**: 用户反馈（截图）

### 问题描述
首页"文档导航"区域的表格样式混乱：
- 表格边框显示异常
- 内容挤在页面左下角
- Markdown 表格未正确渲染

### 可能原因
- VitePress 版本过老（v1.0.0-alpha.28）
- alpha 版本对表格和自定义容器支持不完整

### 解决方案
1. 升级 VitePress 到最新稳定版（v1.x 或 v2.x）
2. 检查 `::: tabs` 语法兼容性
3. 验证表格渲染
