- OpenClaw 定位为**自托管网关**，连接聊天应用到 AI 助手
- 支持 Node 24（推荐）或 Node 22.16+ LTS
- Pi 是唯一内置支持的 coding agent
- 多通道支持：WhatsApp, Telegram, Discord, iMessage 等
- 移动端节点：iOS/Android 支持 Canvas、相机、语音唤醒

**安装流程**（官方推荐）:
```bash
# macOS/Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows PowerShell
iwr -useb https://openclaw.ai/install.ps1 | iex

# 运行配置向导
openclaw onboard --install-daemon

# 打开控制面板
openclaw dashboard
```
