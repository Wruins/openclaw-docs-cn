### 🔥 重大变更

#### 1. Agent 运行时变更
**影响等级：高**

- **Pi 是唯一支持的 coding agent**
- Legacy 路径已移除：Claude、Codex、Gemini、Opencode
- 所有文档和示例需要更新为使用 Pi

**配置示例更新**:
```json5
{
  // 旧配置（已废弃）
  // agents: { codex: {...}, claude: {...} }
  
  // 新配置
  agents: {
    pi: {
      enabled: true,
      // Pi 配置...
    }
  }
}
```

#### 2. 安装方式更新
**影响等级：中**

- **推荐安装方式**：重新运行官网安装脚本
  - macOS/Linux: `curl -fsSL https://openclaw.ai/install.sh | bash`
  - Windows: `iwr -useb https://openclaw.ai/install.ps1 | iex`
- 安装脚本自动检测现有安装并升级
- 可选 `--no-onboard` 参数跳过向导

#### 3. 自动更新器（新功能）
**影响等级：中**

新增可选的自动更新功能（默认关闭）:

```json5
{
  update: {
    channel: "stable",  // stable | beta | dev
    auto: {
      enabled: false,   // 默认关闭
      stableDelayHours: 6,
      stableJitterHours: 12,
      betaCheckIntervalHours: 1
    }
  }
}
```

### 🔧 配置变更

#### 新增配置项

1. **Tailscale 集成**
```json5
{
  gateway: {
    tailscale: {
      mode: "off",    // off | serve | funnel
      // serve: 仅 tailnet 访问
      // funnel: 公开访问（需要密码认证）
    }
  }
}
```

2. **更新检查**
```json5
{
  update: {
    checkOnStart: true,  // 启动时检查更新（默认 true）
  }
}
```

3. **DM 安全策略**
```json5
{
  channels: {
    discord: {
      dmPolicy: "pairing",  // pairing | open
      // pairing: 未知用户需要配对码（默认，安全）
      // open: 允许所有 DM（需要显式配置 "*"）
    }
  }
}
```

### 🛠️ 命令变更

#### 推荐命令流程

```bash
# 安装/升级
curl -fsSL https://openclaw.ai/install.sh | bash

# 或 Windows PowerShell
iwr -useb https://openclaw.ai/install.ps1 | iex

# 运行向导（新安装）
openclaw onboard --install-daemon

# 添加渠道
openclaw channels add

# 检查状态
openclaw gateway status
openclaw logs --follow

# 更新
openclaw update --channel stable

# 诊断
openclaw doctor
openclaw health

# 配对管理
openclaw pairing list
openclaw pairing approve <channel> <code>
```

### ⚠️ 安全提醒

1. **DM 配对策略**：默认 `pairing` 模式，未知用户需要配对码
2. **运行 `openclaw doctor`**：定期检查配置安全性
3. **凭证保护**：App Secret 等敏感信息不要提交到版本控制
4. **远程访问**：使用 Tailscale Serve 而非公开暴露
