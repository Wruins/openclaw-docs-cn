## 🛠️ 开发环境

### 前置要求

- Node.js 18+
- Git
- 代码编辑器（VS Code 推荐）

### 源码安装

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm run build
```
